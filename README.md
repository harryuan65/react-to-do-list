# To-do List Client - React

This is a simple to-do list using React & Typescript, created with create-react-app.

[Demo on github pages](https://harryuan65.github.io/to-do-list-react/)

## Environment

| env  | version  |
| ---- | -------- |
| node | v16.15.1 |
| yarn | 1.22.19  |

## Development

```bash
yarn install
yarn start
```
